### 👍 Practice Questions with Solutions

**Practice:**
Write a program to input a number and print if it’s positive, negative, or zero.

**Solution:**

```python
number = int(input("Enter a number: "))
if number > 0:
    print("Positive 😊")
elif number < 0:
    print("Negative 😞")
else:
    print("Zero 😐")
```
